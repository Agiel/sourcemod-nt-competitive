sourcemod-nt-competitive
========================

SourceMod plugin for competitive Neotokyo. WIP.

Features:
  - Automatic demo recording.
  - Automatic SourceTV recording
  - Count rounds, announce winner.
  - Fade to black
  - Handle dis/reconnecting players and teams
  - Pausing system.
  - Ready up system.
